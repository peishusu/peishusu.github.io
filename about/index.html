项目经验

逸品出行 2023.01 - 2023.07 后端开发人员

项目描述：

  • 逸品出行是一个在线打车平台，司机入驻该平台后可以提供叫车服务。项目包含三个角色：乘客端、司机端、BOSS端，乘客端具有

    进行注册登录、预估价格、下单、接受派单、支付订单等功能，司机端具有注册登录、听单、抢单、订单流转等功能，BOSS端具有

    乘客管理、司机管理、车辆管理等功能。

  • 技术框架：

    • SpringBoot + Spring Cloud Alibaba + Nacos + Redis + Mysql + XXL-JOB + Seata

  • 职责描述：

    • 前期需求调研，依据网约车国家标准文件，制定精细化落地方案，实现业务逻辑完整闭环。

    • 制定并分配任务，搭建基础开发环境。

    • 基于XXL-JOB实现分布式订单调度，保证可用性实现最终数据一致性。

    • 基于Redis作为缓存，来存放乘客、司机登录时分发的验证码以及登录之后的双token。

    • 基于JWT进行token的分发和验证，实现对乘客、司机的权限控制以及免登录。

    • 基于Redission实现分布式锁，引入主从复制和哨兵模式，防止单点故障，实现不同司机抢单数据一致性的保证

    • 基于SpringCloud Alibaba Seata实现分布式事务，保证不同微服务之间事务的原子性和一致性。

    • 基于SpringCloud Alibaba Sentinel实现限流方案，预防雪崩现象。

    • 整合高德地图的猎鹰服务API，实现LBS服务。
